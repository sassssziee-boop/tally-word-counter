# Tally — Word & Character Counter

A clean, free, live word and character counter. Type or paste text and instantly see word count, character count (with/without spaces), sentence count, paragraph count, line count, longest word, average word length, and estimated reading time.

**Live tool:** _add your Vercel URL here after deploying_

## Why I built this

I've personally needed this constantly — checking a bio against a character limit, trimming a caption, making sure an essay or cover letter stays under a word count before submitting it somewhere. Most online counters are cluttered with ads or require sign-up. This one is a single page, works instantly, and nothing you type ever leaves your browser.

## Tech

Plain HTML, CSS, and vanilla JavaScript — no framework, no build step, no dependencies. Fonts are loaded from Google Fonts (Fraunces + JetBrains Mono).

## Run locally

Just open `index.html` in any browser. No build step needed.
